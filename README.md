# Spring ORM - JPA CRUD Using EntityManager

## Overview
This project is a simple Practice POC (Proof of Concept) created to understand and implement CRUD operations using Spring ORM with JPA and the EntityManager API.

The project focuses on learning how JPA works internally with Spring and how EntityManager can be used directly for database interactions instead of using Spring Data JPA repositories.

---

## Features
- Create records in database
- Read/Fetch records
- Update existing records by phoneNumber
- Delete records by phoneNumber
- Entity mapping using JPA annotations
- Transaction management
- Usage of EntityManager for persistence operations

---

## Technologies Used
- Java 17
- Spring Framework
- Spring ORM
- JPA (Jakarta Persistence API)
- Hibernate
- Maven
- MySQL

---

## Concepts Practiced
- EntityManager methods
  - persist()
  - merge()
  - remove()
  - find()
- Persistence Context
- Entity Lifecycle
- Transaction Handling
- ORM Mapping
